Gestione Sito 

In questa pagina vi sono 3 pulsanti che reinderizzano a tre campi di gestione,

1) Gestione Report : ti consete di otenere un file con tutti i dati relativi al sito consultabile ofline, oppure puoi scegliere se ottere un file che contenga dei dati specifici per le singole categorie.

2) Gestione Raccolta : ti permettere di gestire i prodotti presenti sul sito in base alla loro tipologia 

3)Gestione Utenti : ti permette di gestire gli utenti registrati al sito e modificarne i dati o i livelli di autorizzazione
